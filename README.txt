Ce projet décrit un script Arduino pour Arduino Mega, interfaçant un capteur de gaz MQ4 (méthane) vers un bargraph LED à 5 niveaux.
Des options ont été ajoutées (délai de préchauffage, BP de contrôle des LEDs, LED indiquant le niveau de détection de seuil TR de la carte capteur.

This project describe an arduino script write for the Arduino Mega, interfacing the MQ4 sensor (methane sensor) to a LED bargraph, 5 levels.
Options are put on the project (preheat delay, LED control push button, remote LED for the sensor threshold detector.
